# Trigger Model

Authority editing requires a trigger τ.

```text
τ = conflict | error | external perturbation | closure precursor
```

## Function

τ activates re-entry governance.

Without τ, authority may remain unchallenged until revision cost exceeds update budget.

## Diagnostic question

Is conflict treated as editing material or as a threat to be suppressed?

If conflict is suppressed, closure risk increases.

If conflict triggers minimal editing, open persistence can be maintained.
