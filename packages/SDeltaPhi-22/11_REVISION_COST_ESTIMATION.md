# Revision Cost Estimation

AEP compares revision cost and continuation cost.

```text
RevisionCost(A) vs ContinuationCost(A)
```

## Revision is possible when

```text
UpdateBudget ≥ RevisionCost(A)
```

## Closure risk rises when

```text
RevisionCost(A) > UpdateBudget
```

## Diagnostic question

Is it cheaper to minimally edit authority now, or to continue under current authority until closure debt increases?
