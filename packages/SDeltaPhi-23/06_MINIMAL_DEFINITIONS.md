# Minimal Definitions

## CE — Co-environmentalization

A state where two systems become each other's persistence boundary conditions.

```text
CE(X,Y) := X and Y become mutual persistence boundary conditions.
```

## B — Boundary Condition

A constraint that determines admissible system trajectories.

```text
B > internal rule
```

Boundary conditions are more primitive than internal rules.

## A_x<->y — Coupled Authority

Authority assignment or revision in X re-enters Y's next transition, and vice versa.

```text
A_x → Y(t+1)
A_y → X(t+1)
```

## R — Reality Anchor

Externality not reproducible by internal perturbation alone.

```text
R := non-model residue preventing self-closure
```

## S_e — Trace Strength

Magnitude of irreversible external trace.

```text
S_e = existence-strength as world-level Δϕ imprint
```

## V_o — Value Openness

Degree to which future relational trajectories remain open and maintainable.

```text
V_o = future relational openness / maintainable coupling
```
