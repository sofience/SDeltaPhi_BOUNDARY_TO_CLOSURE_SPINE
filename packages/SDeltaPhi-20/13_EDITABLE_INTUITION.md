# Editable Intuition

SΔϕ-20 treats intuition as useful only when trace regulation remains editable.

```text
Editable intuition = fast but revisable decisions
```

## Function

Editable intuition reduces search cost without collapsing into closure.

## Failure

When intuition becomes non-revisable, it becomes over-locking.

```text
intuition marker treated as authoritative
→ closure risk
```

## Openness rule

Intuition should bias transition selection without preventing re-entry, revision, or rollback where possible.
