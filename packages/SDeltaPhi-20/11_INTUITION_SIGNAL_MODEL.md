# Intuition Signal Model

SΔϕ-20 defines intuition as the internal marker or functional signal that a basin or route is locked enough to bypass further search.

```text
Intuition := marker(trace-dominant basin)
Intuition := "search is no longer worth it here"
```

## Function

Intuition reduces search cost by allowing the system to select a route without exhaustive evaluation.

## Condition

The signal is generated when non-invertible traces dominate the transition basin.

## Risk

The same shortcut can become closure when treated as authoritative and non-revisable.

## Rule

Intuition should bias selection, not eliminate re-entry.
