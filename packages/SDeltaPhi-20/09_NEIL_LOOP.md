# No-Exit Intuition Loop — NEIL

NEIL summarizes how intuition emerges as a maintenance-friendly shortcut.

```text
Input stream
(external / internal)
        ↓
Compression
(many → one)
        ↓
No-exit test
(rollback cost ≫ budget?)
        ↓
Trace fixation
(non-invertible residue)
        ↓
Intuition signal I
("skip search here")
        ↓
Next selection / action
(biased transition)
        ↺
Re-entry
```

## Function

Compression produces traces.

Traces bias next selection.

The bias is felt or used as intuition.

## Boundary

NEIL does not prove shared meaning.

NEIL describes constraint-driven trace re-entry.
