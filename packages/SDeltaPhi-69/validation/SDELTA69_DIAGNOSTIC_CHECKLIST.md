# SΔϕ-69 Diagnostic Checklist

Use this checklist before applying SΔϕ-69.

- [ ] Identify the prior state.
- [ ] Identify the non-zero transition.
- [ ] Identify the restored or apparently restored surface state.
- [ ] Separate surface equivalence from transition-history erasure.
- [ ] Estimate TCC if restoration cost matters.
- [ ] Identify residual irreversibility.
- [ ] State uncertainty and limits.
- [ ] Avoid single-cause historical reduction.
- [ ] Avoid treating all restoration as fake.
- [ ] Avoid moralizing structural diagnosis.

## Minimal conclusion template

```text
The case shows [surface restoration / symbolic reversal / failed restoration].
However, the transition history includes [X], and residual irreversibility remains in [Y].
Therefore the apparent return should be treated as [diagnosis label], not as non-occurrence.
```
