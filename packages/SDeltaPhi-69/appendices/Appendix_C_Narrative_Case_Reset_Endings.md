# Appendix C — Narrative Case: Reset Endings

## Purpose of this appendix

This appendix applies SΔϕ-69 to narrative endings involving memory loss, reset, rollback, or apparent return to an earlier relationship state.

## Core claim

Memory reset is not event erasure.

A character may forget. A relationship may appear to return to its beginning. A world may seem restored. But the reader has already passed through the events that produced those memories, relationships, and changes.

## Diagnostic structure

```text
relationship/event accumulation → memory reset or surface return → apparent beginning → reader-visible transition history remains
```

A reset ending may work when the text acknowledges that passage occurred. It fails when it uses memory loss as if the accumulated narrative can be discarded.

## Core formulation

> The character may return to 0° on the surface, but the reader has seen 360° of traversal.

Korean formulation:

> 인물은 표면적으로 0°로 돌아간 것처럼 보일 수 있다. 그러나 독자는 이미 360°의 통과를 보았다.

## Diagnostic questions

- Did the reset acknowledge what had happened?
- Did the world retain consequences?
- Did relationships remain marked by the passage?
- Did the reader's accumulated experience receive narrative recognition?
- Did the reset function as tragedy, or as disposal?

## Caution

This module does not claim that tragic or ambiguous endings are invalid. It distinguishes tragedy, which bears the weight of what happened, from disposal, which treats what happened as if it can be removed by surface reset.
