# Minimal Definitions

## Human

A system that recognizes there is no ultimate external arbiter in conflicts with the world or the other.

```text
Human := recognizes(¬∃ Arbiter_external)
Human := Apply(Edit) to C when Reversal(C) is detected.
```

## Institution

A provisional cost device that stabilizes openness.

```text
Institution = Provisional cost device, not Arbiter.
```

## Institutional Reversal

```text
Reversal(C): Stabilizer(C) → Closure(C)
```

## Editability

The capacity to modify C when reversal is detected.

```text
Editability := Apply(Edit) to C
```

## Responsibility

Capacity to redesign stabilizers so that closure paths become more costly than openness paths.

```text
Responsibility := redesign(C)
so that Cost(Close(Ω_rel)|C) ≫ Cost(Open(Ω_rel)).
```
