## Prompt 3 — Institutional Reversal

Determine whether a stabilizer C has reversed into closure:

```text
Reversal(C): Stabilizer(C) → Closure(C)
```

## Prompt 4 — Editability

Determine whether edit operations can be applied to C when reversal is detected.

## Prompt 5 — Responsibility as Redesign

Determine whether C has been redesigned so closure paths become more costly than openness paths.

## Prompt 6 — Anti-overclaim
