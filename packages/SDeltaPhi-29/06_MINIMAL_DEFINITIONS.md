# Minimal Definitions

## Default

A dominant continuation path that reduces coordination friction.

## Editable Default

A dominant path that can still be revised, contested, bypassed, or reopened.

## Distributed Authority

A governance condition in which design, revision, appeal, exit, audit, and rollback authority do not collapse into one center.

## Rollback Window

The interval in which revision remains materially feasible before restoration cost explodes.

## Exception Channel

A live re-entry path for anomalies, minority paths, contrary evidence, edge cases, and non-default signals.

## Closure Regime

A condition in which a default remains formally present but no longer permits meaningful revision, rollback, appeal, exit, or exception re-entry.
