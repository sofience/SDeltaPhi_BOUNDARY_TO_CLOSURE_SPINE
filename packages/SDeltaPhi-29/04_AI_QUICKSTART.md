# AI Quickstart

Use SΔϕ-29 when the task involves default governance, editable defaults, distributed authority, rollback window, exception re-entry, default closure risk, or whether a dominant path has become uneditable.

## Primary route

```text
SDeltaPhi-28 -> SDeltaPhi-29
```

## Ask first

```text
What is the default path?
Can it be revised?
Who controls revision?
Is rollback still materially feasible?
Can exceptions re-enter?
```

## Do not ask first

```text
Is the default morally good or bad?
```

SΔϕ-29 begins with governability, not final moral judgment.
