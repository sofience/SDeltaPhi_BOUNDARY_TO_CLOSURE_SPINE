# Editable Default

A default can coordinate action without becoming closed power.

Editable Default names a dominant path that remains revisable, contestable, bypassable, or reopenable.

```text
Editable Default =
dominant path
+ still revisable
+ still contestable
+ still bypassable
+ still reopenable
```

A default is not editable merely because a rule says it can be changed.

It is editable only when revision remains materially feasible.
