# Case Pattern — Mandatory AI Image Filtering as Editable Default Test

## Case type

Public policy / platform governance / technical filtering default.

## Structural framing

A legal or platform system makes AI-based image filtering the default path for image publication or distribution.

SΔϕ-29 does not begin by declaring the default censorship or legitimacy.

It detects default-governance signals.

## SΔϕ-29-relevant signals

```text
mandatory default path
authority concentration risk
rollback-window sensitivity
exception-channel dependency
false-positive and false-negative restoration burden
```

## Four-condition audit

```text
Ed: Is the filtering default editable and contestable?

Ad: Is authority over design, review, appeal, technical evaluation, and audit distributed?

Wr: Does rollback remain materially feasible after false positives or false negatives?

Ce: Can public-interest, journalistic, educational, evidence-preservation, artistic, or contextual exceptions re-enter review?
```

## Boundary

This case pattern is not a blanket claim that AI image filtering is censorship.

It is not a blanket claim that filtering is legitimate.

It is a structural test of whether a mandatory technical filtering default remains governable.
