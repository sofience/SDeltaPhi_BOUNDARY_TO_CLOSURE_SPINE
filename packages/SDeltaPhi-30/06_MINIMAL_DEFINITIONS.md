# Minimal Definitions

## Declared editability

A system states that revision, appeal, participation, exception handling, or rollback is available.

## Operational editability

Revision, rollback, exception re-entry, and authority distribution actually modify the default path.

## Revision Reality

The default is actually revised, not merely declared revisable.

## Rollback Reality

Rollback remains timely while restoration cost is still manageable.

## Exception Re-entry

Non-default signals can materially alter the dominant path.

## Authority Distribution

Multiple actors can materially alter the default path, not merely advise.
