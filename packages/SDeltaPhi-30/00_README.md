# SΔϕ-30 AI-READABLE Package — Diagnosing Closed Defaults (v1.0)

This package provides the AI-readable structured companion package for **SΔϕ-30 — Diagnosing Closed Defaults: Minimal Tests for Editable Governance after SΔϕ-29 (v1.0)**.

Package DOI: `https://doi.org/10.5281/zenodo.20362950`

## Core thesis

Editability claimed is not editability real.

A system is editable only when revision is exercised, rollback remains timely, exceptions can materially re-enter, and authority is distributed in a way that can actually modify the default path.

## Core formula

```text
D* editable-real iff Rv ∧ Rb ∧ Ex ∧ Ad
```

## Core tests

```text
Revision Reality Test
Rollback Reality Test
Exception Re-entry Test
Authority Distribution Test
```

## Role

SΔϕ-30 extends SΔϕ-29 by turning editable default conditions into operational diagnostic tests.

## Do not use this package as

- a claim that every stable default is closed;
- a moral judgment engine before testing editability;
- a replacement for SΔϕ-56 when measuring rollback cost;
- a claim that feedback collection equals exception re-entry;
- a claim that consultation equals distributed authority.
