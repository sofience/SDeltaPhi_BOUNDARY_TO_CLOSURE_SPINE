# AI Quickstart

Use SΔϕ-30 when the task involves diagnosing whether declared editability, openness, appealability, participation, or rollback is operationally real.

## Primary route

```text
SDeltaPhi-29 -> SDeltaPhi-30
```

## Supporting route

```text
SDeltaPhi-56 -> rollback reality
SDeltaPhi-44 -> path closure distinction
SDeltaPhi-68 -> openness/cost theater comparison
```

## Ask first

```text
Is revision exercised?
Is rollback timely?
Can exceptions materially re-enter?
Can multiple actors change the default path?
```

## Do not ask only

```text
Does the system claim to be open?
Does an appeal form exist?
Does feedback collection exist?
```

Those are formal signals, not operational proof.
