# Rollback Reality Test

Question:

```text
Does rollback remain possible while restoration cost is still manageable?
```

Formula:

```text
t < t*_r → rollback operational
t ≥ t*_r → restoration cost rises sharply
```

Core distinction:

```text
formal reversibility
≠
operational rollback
```

Rollback exists operationally only while it can still be enacted before fixation becomes too expensive.
