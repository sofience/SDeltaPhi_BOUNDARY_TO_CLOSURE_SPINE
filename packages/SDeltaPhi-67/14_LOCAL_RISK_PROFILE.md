# Local Risk Profile

A local risk profile describes how much damage can occur if routing fails.

## Risk dimensions

```text
domain risk
user harm potential
external-world write access
rollback capacity
verification burden
restoration cost exposure
authority sensitivity
privacy sensitivity
legal sensitivity
medical sensitivity
financial sensitivity
```

## Rule

Higher risk requires:

```text
stronger evidence binding
lower escalation threshold
more audit logging
more human review
stricter output-level control
```

## Boundary

Risk profile is not a moral label.

Risk profile is a routing and threshold condition.
