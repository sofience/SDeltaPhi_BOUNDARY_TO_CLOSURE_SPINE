# Cost Theater

Cost Theater occurs when visible cost appears low while the real transition completion cost is hidden, delayed, displaced, or externalized.

## Formula

```text
Cost Theater =
C_visible ↓
while
TCC_total remains high, hidden, delayed, or externalized
```

## Core pattern

```text
surface:
cheap / free / accessible / open

hidden terrain:
verification cost
repetition cost
context loss
tool access limits
failure correction
restabilization burden
infrastructure cost
self-location uncertainty
```

## Boundary

Cost Theater does not mean all cheap access is deceptive.

It means low visible price must not be confused with low completion cost.
