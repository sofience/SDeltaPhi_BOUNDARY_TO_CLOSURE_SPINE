# Brand-Level Capability Opacity

Brand-Level Capability Opacity occurs when access to a brand name is mistaken for access to the brand's full capability map.

## Formula

```text
Access to intelligence surface
≠
Access to actual capability layer
```

## Pattern

```text
The user says:
"I am using Claude."
"I am using GPT."
"I am using Gemini."

SΔϕ-68 asks:
Which model layer?
Which route?
Which tools?
Which limits?
Which gated capabilities?
Which product-family map?
```

A brand can be a surface.

A capability layer is a route.

Do not confuse them.
