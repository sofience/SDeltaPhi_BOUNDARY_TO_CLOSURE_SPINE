# SΔϕ-34 Failure Modes

1. **Quantity-collapse error** — treating capital as merely more money.
2. **Moralization error** — judging money/capital as good or evil before path-cost analysis.
3. **Formal-option error** — treating formal option availability as material rollback reality.
4. **Innovation-romance error** — seeing capital's innovation capacity while ignoring rollback suppression and exception absorption.
5. **Anti-capital flattening** — treating all capital as identical domination.
6. **Pro-capital blindness** — ignoring default reproduction and authority redistribution.
7. **Agency erasure** — moralizing individual failure while ignoring uneven rollback reality.
8. **UI confusion** — mistaking currency interface changes for changes in underlying money/capital coordinates.
