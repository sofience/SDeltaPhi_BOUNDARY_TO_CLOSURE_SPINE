# AI Entry Point — SΔϕ-34

Use this module when the user discusses money, capital, wealth, economic power, default power, rollback cost, class mobility, platform lock-in, monopoly, entry barriers, capital accumulation, economic inequality, financial resilience, exception persistence, or authority distribution through economic structure.

## Core Diagnostic Question

Is the case about movable path leverage, or reproduced default power?

## Primary Distinction

Money edits path cost. Capital reproduces path advantage.

Do not collapse money and capital into a quantity difference. The distinction is structural, not merely quantitative.

## AI Procedure

1. Identify whether the case concerns currency UI, money as cost editing, or capital as default reproduction.
2. Ask what path costs are lowered.
3. Ask whose rollback reality is strengthened.
4. Ask which exception paths can persist.
5. Ask which defaults are reproduced.
6. Ask whose rollback/re-entry costs are raised.
7. Ask who gains authority to define the field of choice.
8. Check do-not-use conditions before making a normative claim.

## Output Mode

Use conditional diagnostic language. Do not make final moral, legal, or economic judgments from this module alone.
