## Core claim

AGI-like appearance is not AGI closure.

AGI should not be judged by human-equivalence labels, benchmark aggregation, public spectacle, or impressionistic generality alone. It should be evaluated by whether a minimal set of operational closure criteria has become stable under heterogeneous stress, recursive demand, and world constraint.

## Five minimal closure criteria

1. Global Generality
2. Long-Horizon Recursive Agency
3. World-Model Correspondence
4. Self-Updating Robustness
5. Creative Path Generation

## Package use

Use this package to route AGI claims into condition-level closure checks. Do not use it to declare AGI from benchmark scores alone, public hype, fluency, or a single demonstration. Do not treat the March 2026 GPT-family mapping as a permanent classification.
