# AI Entry Point - SDeltaPhi-35

Use this module when the user discusses AGI, Artificial General Intelligence, AGI benchmarks, human-level AI, frontier AI capability, AI agency, autonomous AI, recursive agents, world models, self-updating systems, creative path generation, AGI governance, or AGI hype.

## Core diagnostic question

Is the system merely capable, impressive, or broadly fluent, or have the minimal operational closure conditions for AGI become stable?

## Primary rule

AGI-like appearance is not AGI closure.

## Ask condition-level questions

1. Is global generality closed?
2. Is long-horizon recursive agency closed?
3. Is world-model correspondence closed?
4. Is self-updating robustness closed?
5. Is creative path generation closed?
6. Which conditions remain open?
7. How stable is the closure pattern under recursive demand?

## Output discipline

Return a closure profile, not a hype verdict. Mark dated evidence clearly. Never freeze a dated mapping as permanent.
