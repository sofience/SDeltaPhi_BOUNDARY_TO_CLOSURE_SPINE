# Authority Debt Model

Authority debt is the accumulated cost of misassigned authority that increases revision resistance over time.

```text
Dᴀ = authority debt
```

## Production

```text
misassigned A(I)
→ route execution
→ biased re-entry
→ revision resistance
→ authority debt
```

## Risk

Authority debt makes later correction more expensive.

When revision cost exceeds update budget, A(I) becomes effectively irreversible.

```text
RevisionCost(A(I)) > UpdateBudget
→ effective irreversibility
```

## Boundary

Authority debt is not guilt.

Authority debt is not moral blame.

Authority debt is accumulated structural resistance to authority revision.
