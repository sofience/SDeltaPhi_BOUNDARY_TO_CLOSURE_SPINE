# Authority Assignment Model

SΔϕ-21 defines judgment as authority assignment over intuition markers.

```text
Judgment := J: I → A(I)
```

## Meaning

The system does not act on all intuition markers equally.

It assigns priority, gates alternatives, and executes one route.

That assignment is judgment.

## Function

```text
multiple intuition markers
→ authority assignment
→ executable route
```

## Boundary

Authority is not truth.

Authority is not moral certainty.

Authority is not legal legitimacy.

Authority is a route-selection weight or gate.
