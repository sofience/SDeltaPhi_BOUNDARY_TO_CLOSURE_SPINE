# Judgment Axioms

## Axiom 21-1 — Intuition Plurality

Any bounded system generates multiple intuition markers across competing basins.

```text
I₁ … Iₙ
```

## Axiom 21-2 — Authority Assignment

Judgment assigns authority A to a subset of intuition markers, producing an executable route.

```text
J: I → A(I)
```

## Axiom 21-3 — Irreversibility Threshold

If revision cost exceeds the update budget, authority becomes effectively irreversible.

```text
RevisionCost(A(I)) > UpdateBudget
→ A(I) becomes effectively irreversible
```

## Axiom 21-4 — Two Failure Modes

```text
Over-authority → closure.
Under-authority → drift.
```

## Axiom 21-5 — Open Persistence

Sustained openness requires periodic authority re-editing triggered by conflict, error, or external perturbation.

```text
Open persistence requires re-editable A(I).
```

## Boundary

These axioms are substrate-agnostic.

They apply to individual cognition, organizations, and AI systems as long as multiple basins exist and authority can become costly to revise.
