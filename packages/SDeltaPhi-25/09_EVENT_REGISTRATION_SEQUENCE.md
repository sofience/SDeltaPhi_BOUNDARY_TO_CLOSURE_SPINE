# Event Registration Sequence

SΔϕ-25 distinguishes difference, event, trace, and structural consequence.

```text
Pre-detected difference
-> threshold passage
-> event registration
-> irreversible trace
-> re-entry into subsequent state
-> structural consequence
```

## Core distinction

Not every difference becomes historical.

Not every variation becomes ontological.

Only differences that cross the threshold of registration and re-enter future state formation become irreversibly consequential.
