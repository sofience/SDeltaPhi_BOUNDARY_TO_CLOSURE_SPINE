# Pre-detection Axioms

## Axiom 25-1 — Difference May Operate Prior to Registration

A difference may affect system dynamics before it is recognized, named, measured, or institutionally acknowledged.

```text
operativity != recognition
```

## Axiom 25-2 — Detection Requires Threshold Crossing

Detection occurs only when a latent difference crosses the threshold sufficient for event registration within the current interpretive frame.

```text
latent difference + threshold crossing -> detection
```

## Axiom 25-3 — Irreversibility Begins at Registration

A difference becomes irreversibly consequential only when it is registered as an event and re-entered into later state formation.

```text
registration + re-entry -> irreversible consequence
```

## Axiom 25-4 — Authority Governs Detection Bandwidth

Every authority regime determines which differences are detectable, ignorable, suppressible, or unspeakable.

```text
authority -> detection bandwidth
```

## Axiom 25-5 — Innovation Emerges from the Pre-detected

Innovation occurs when a previously pre-detected difference becomes registrable under a revised frame.

```text
pre-detected difference + revised frame -> innovation
```
