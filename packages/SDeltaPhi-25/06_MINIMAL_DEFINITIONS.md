# Minimal Definitions

## Pre-detection

The state in which a difference is already operative within a system but has not yet crossed the threshold required for event registration.

```text
Pre-detection = latent operativity without event registration
```

## Latent Difference

A difference that alters transition tendencies, interpretive sensitivity, or future threshold conditions before being recognized.

```text
latent difference = active but not yet registered
```

## Detection

Threshold crossing into registrability.

```text
d_latent >= theta_frame -> detection
```

## Event Registration

The moment a difference becomes recognized as an event, trace, or structurally consequential state.

```text
threshold crossing -> event registration
```

## Irreversible Trace

A registered difference that re-enters subsequent state formation.

```text
event registration + re-entry -> irreversible trace
```

## Detection Bandwidth

The range of differences a frame or authority regime can perceive, name, measure, or allow to appear.

```text
detection bandwidth = what can appear as event
```

## Structural Invisibility

A condition where operative differences remain outside the current frame's detection bandwidth.

```text
operative but frame-invisible
```
