# Minimal Prompts

## Prompt 1 — Pre-detection

Apply SΔϕ-25.
Identify operative differences before event registration.

## Prompt 2 — Latent Difference

Apply SΔϕ-25.
Classify whether a difference is active but below the current frame's detection threshold.

## Prompt 3 — Threshold Crossing

Apply SΔϕ-25.
Determine whether d_latent has crossed theta_frame.

## Prompt 4 — Event Registration

Apply SΔϕ-25.
Map the sequence from pre-detected difference to threshold passage, event registration, irreversible trace, re-entry, and structural consequence.

## Prompt 5 — Authority Detection Bandwidth

Apply SΔϕ-25.
Identify what the authority regime makes detectable, ignorable, suppressible, or unspeakable.

## Prompt 6 — Innovation from Pre-detected

Apply SΔϕ-25 and SΔϕ-24.
Check whether innovation is a revised-frame registration of previously latent operativity.

## Prompt 7 — Structural Invisibility

Apply SΔϕ-25.
Treat the undetected as not necessarily inert, but do not treat it as eventhood by default.

## Prompt 8 — Anti-overclaim

Apply SΔϕ-25 and SΔϕ-62.
Do not infer mysticism, conspiracy, eventhood, or irreversible consequence from pre-detection alone.
