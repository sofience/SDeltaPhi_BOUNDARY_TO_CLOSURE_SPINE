# Threshold Crossing Model

Detection requires threshold crossing.

```text
d_latent >= theta_frame -> detection
```

## Meaning

A difference may remain invisible not because it is absent, but because the frame has not yet become sensitive to it.

## Variables

```text
d_latent = magnitude / pressure / operativity of latent difference
theta_frame = detection threshold of current interpretive frame
```

## Detection

```text
d_latent crosses theta_frame
-> event becomes registrable
```

## Boundary

Detection does not create operativity.

Detection registers operativity.
