# Diagnostic Regimes

## Latent Operativity

```text
difference active
but not yet detectable
```

Risk:

```text
false absence judgment
```

## Suppressed Detection

```text
difference would be detectable
but authority marks it ignorable / unspeakable
```

Risk:

```text
delayed crisis
```

## Threshold Passage

```text
latent difference crosses theta_frame
event becomes registrable
```

Risk:

```text
sudden-event illusion
```

## Event Registration

```text
difference becomes recognized as event / trace / state
```

Risk:

```text
registration mistaken for origin
```

## Innovation Registration

```text
previously pre-detected difference becomes registrable under revised frame
```

Result:

```text
innovation becomes structurally intelligible
```
