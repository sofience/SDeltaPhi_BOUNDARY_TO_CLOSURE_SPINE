This package provides an AI-readable decomposition of SΔϕ-71: Cost Attribution Redesign. It defines cost attribution redesign as a process for reducing transition cost without hiding that cost in specific actors, groups, systems, or environments.

## Core Claims

- Cost attribution redesign is not merely a fairness or equality slogan.
- Low-cost transition is invalid when achieved through hidden asymmetric burden.
- Nominal equality does not guarantee TCC equality.
- Human TCC and AI TCC must be distinguished.
- Unsettled AI subjecthood is not an exemption from cost audit.
- Binding Fiction is required for shareable redesign.
- Editing Fiction is required to keep redesign from becoming closure.

## Key Formulae

```text
Low-Cost Redesign ≠ Cost Externalization
Human_TCC(task) ≠ AI_TCC(task)
Subjecthood_Unsettled ≠ Audit_Exemption
Nominal Equality ≠ TCC Equality
Legal Validity ≠ Cost Symmetry
```

## Use This Package For

- cost attribution redesign
- asymmetric cost burden
- TCC decomposition
- AI/human task-cost comparison
- taxation as institutional cost attribution redesign
- public-service conscription as legal cost asymmetry
- fairness/equality failure-mode analysis
- hidden verification/repair/re-entry cost diagnosis

## Do Not Use This Package For

- replacing legal, economic, labor, policy, or clinical analysis
- claiming all costs must be equal
- claiming AI is already a full subject
- claiming AI can never be audit-relevant
- moralizing individuals while ignoring structural burden
- erasing individual responsibility by blaming structure alone
