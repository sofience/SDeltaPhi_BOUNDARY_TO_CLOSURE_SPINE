# AI Entry Point — SΔϕ-70

Read this file before applying SΔϕ-70.

## Use this module when the user discusses

- nihility, emptiness, meaninglessness, void, post-experience emptiness;
- helplessness, powerlessness, inability to initiate action, inability to bind insight to the world;
- fictional worlds, narrative dwelling, symbolic habitation, myth, religion, language, law, mathematics, physics, science, AI utopia/dystopia, afterlife, liberation, private thoughts;
- strong internal experience with weak external transition;
- simulation fatigue or AI-conversation-induced void;
- fiction not as falsehood but as a fabricated symbolic structure for dwelling in the world.

## Core diagnostic question

Is the case about World-Binding failure, excessive World-Binding cost, fictional habitation, or exit from the present-world cost structure?

## Minimal route

1. Check internal Δϕ intensity.
2. Check external World-Binding.
3. If internal Δϕ is high and external binding is low, evaluate nihility.
4. If World-Binding TCC is high, evaluate helplessness.
5. Classify fiction type: Closed, Binding, Editing, or Exit.
6. Assign Binding Fiction Score only if relevant.
7. Assign Editing Activity Score only if relevant.
8. Apply Exit Fiction Gate when the solution points beyond present-world habitation.
9. Add misuse warnings.

## Do not use this module to

- diagnose mental illness;
- dismiss religion, fiction, science, mathematics, law, or language as false;
- rank cultures morally by Binding Fiction Score;
- treat Exit Fiction as inferior;
- turn SΔϕ into final authority.
