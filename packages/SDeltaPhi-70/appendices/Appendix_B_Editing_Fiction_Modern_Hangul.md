# Appendix B — Editing Fiction: Modern Hangul

This appendix is an illustrative application, not an exhaustive linguistic history.

## Function in SΔϕ-70

Modern Hangul can be read as Editing Fiction: a Binding Fiction that continues to be revised, standardized, institutionalized, digitized, taught, and technically adapted.

## Interpretation

The transition from Hunminjeongeum to modern Hangul was not a single completed event. It involved scholars, educators, publishers, institutions, orthographic standards, digital input systems, typography, search systems, and AI tokenization environments. Ju Si-gyeong can be treated as a representative figure in this editing process, but the modern Hangul system is the result of broader cumulative editing.
