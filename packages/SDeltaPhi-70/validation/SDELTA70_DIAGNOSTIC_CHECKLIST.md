# SΔϕ-70 Diagnostic Checklist

Use this checklist before producing a SΔϕ-70 diagnosis.

- [ ] Did strong internal Δϕ occur?
- [ ] Did it bind to external action, record, relation, body, institution, shared practice, or cost structure?
- [ ] If not, is nihility a relevant diagnosis?
- [ ] Is World-Binding TCC perceived, learned, or structurally imposed as high?
- [ ] If yes, is helplessness a relevant diagnosis?
- [ ] Is there a fictional or symbolic structure involved?
- [ ] Is it Closed Fiction, Binding Fiction, Editing Fiction, Exit Fiction, or mixed?
- [ ] If score is assigned, is Binding Fiction Score being used descriptively rather than morally?
- [ ] Is Exit Fiction Gate relevant?
- [ ] Have do-not-use conditions been checked?
- [ ] Have failure modes been considered?
